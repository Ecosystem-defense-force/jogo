Ecosystem Defense Force

Um MVP de um jogo tower defense para a cadeira de Sistemas Multimídia

PRAZO FINAL - 9/12


Engine - Godot 4.x

Estamos usando a Godot 4.5.1 

=================================================================================

Ambiente e Versionamento 
  
    -Controle de Versão: Git instalado no sistema.
    
    -Colaboração: Acesso de Colaborador (ou Write Access) ao repositório GitHub.
  
    Setup do Ambiente (via CMD): 
    -Navegue até o Diretório de Projetos: 
        cd /caminho/para/sua/pasta/de/projetos
    -Clone o Repositório: Use o link HTTPS correto do nosso projeto
        git clone [Link HTTPS do Repositório]
    -Acesse a Pasta: Entre na pasta do projeto que foi criada (por exemplo, ./jogo)
        cd jogo

Configuração da Godot

    -Abra o Godot e importe a sua pasta jogo
    -Verifique a resolução: Projeto > Configurações de Projeto > Janela 
        Largura 320 ; Altura 180 
    -Verifique o Scaling: Janela > Esticar 
        Modo Canvas Items ; Aspecto Keep
